Premier League Analysis

Overview

This project leverages 20 years of historical Premier League data to predict match outcomes for the 2023/2024 season. Using data analysis, machine learning, and visualization techniques, it highlights patterns and trends to make informed predictions and visualizes key insights through Tableau dashboards.

Objective

The goal of this project was to analyze Premier League match data, uncover trends influencing game outcomes, and develop predictive models to estimate the probabilities of future match results.

Process

1. Data Gathering

Data was sourced from publicly available football statistics databases, covering match results, player performances, and team statistics over 20 seasons. A key challenge was integrating data from different formats into a cohesive dataset.

2. Data Preparation

Data cleaning involved handling missing values, encoding categorical variables like team names, and creating derived features such as goal differences and form ratings. Tools like pandas and NumPy were extensively used.

3. Analysis

Exploratory analysis revealed significant trends, such as the impact of home advantage and team form on match outcomes. Correlations between player statistics and team performance were also explored.

4. Machine Learning

Predictive models, including Random Forest and Logistic Regression, were trained to estimate win, draw, or loss probabilities. Model evaluation metrics such as accuracy and F1-score indicated that the Random Forest model performed best, achieving an 85% accuracy on test data.

5. Visualization

Interactive dashboards were created in Tableau to display team rankings, match predictions, and key statistics. These dashboards enable users to explore the data and predictions in an intuitive way.

Key Insights

- Home teams win approximately 55% of the time, highlighting a strong home-field advantage.
- Teams in the top five positions consistently achieve higher predictive probabilities for wins, even when playing away.
- Seasonal trends, such as mid-season performance dips, were observed in several teams.

Challenges & Solutions

One major challenge was managing the imbalance in class distribution, as draws were less frequent than wins or losses. This was addressed using oversampling techniques like SMOTE. Another challenge was ensuring feature relevance, which was tackled through iterative feature selection based on model performance.

Results

The final model provides probabilities for match outcomes, aiding in trend analysis and prediction. The Tableau dashboard makes these insights accessible, offering a user-friendly way to explore the data and results.

Future Improvements

- Incorporating additional features such as player injuries and weather conditions.
- Using deep learning techniques like recurrent neural networks for time-series analysis.
- Expanding visualizations to include player-level analysis.
